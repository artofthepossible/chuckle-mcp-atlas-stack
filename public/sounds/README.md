# Sound Effects for ContainerComedy Club

This directory contains sound effects for the application.

## Required Files:
- `drumroll.mp3` - Drum roll sound effect (plays during the 2-second pause)
- `rimshot.mp3` - Rimshot sound effect (plays when punchline is revealed)

## Where to Get Sound Effects:

You can download free sound effects from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [BBC Sound Effects](https://sound-effects.bbcrewind.co.uk/)

## Recommended Searches:
- "drum roll comedy"
- "rimshot"
- "ba dum tss"
- "SNL drum roll"

## File Specifications:
- Format: MP3
- Duration: 2-3 seconds for drum roll, 1-2 seconds for rimshot
- Sample rate: 44.1kHz
- Bit rate: 128kbps or higher

## Installation:
1. Download the sound effects from the sources above
2. Name them `drumroll.mp3` and `rimshot.mp3`
3. Place them in the `public/sounds/` directory
4. The app will automatically use them

## Alternative (Web Audio API):
If you don't want to download files, you can use the Web Audio API to generate simple beep sounds, or use online CDN-hosted sound effects.
